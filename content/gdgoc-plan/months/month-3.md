# Month 3: Skill Cluster Kickoff

Period: Jul 20 - Aug 20, 2026
Phase: Growth

## Strategic Intent
Month 3 converts attention into habit by giving members small communities where they can learn consistently and build confidence.

## Main Objectives
- Launch skill clusters for Frontend, Backend, Design, Data, and AI.
- Begin weekly Saturday sessions.
- Run the first informal hackathon.

## Execution Notes
- Each cluster needs a lead, schedule, learning path, and simple first project.
- Keep sessions practical and demo-heavy.
- Use the mini-hackathon to normalize building, not perfection.

## Milestones
- Five clusters active.
- Weekly sessions started.
- Build in a Day mini-hackathon held.
- 50+ average session attendees.

## Risks To Watch
- Clusters depending on one person only.
- Members feeling left behind if sessions move too fast.
